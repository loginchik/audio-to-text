ПОДГОТОВКА

1. Открыть Terminal

2. В Finder найти папку с этим кодом и скопировать путь к ней. Проще всего это сделать, если нажать на папку правой кнопкой мыши -> зажать option -> Copy "" as Pathname

Он будет примерно таким: /Users/username/path/whisperTrans

3. Переместиться в терминале в эту папку:
>> cd /Users/user/path/whisperTrans

4. Активировать виртуальное окружение, в котором установлены уже все зависимости
>> source venv/bin/activate

Чтобы убедиться, что виртуальные окружение активировано: which python
Команда вернёт путь к питону, с помощью которого будет выполняться следующий код. При хорошем раскладе он должен быть примерно таким: /Users/username/path/whisperTranscriber/venv/bin/python

ЗАПУСК

5. Запустить код
>> python cli.py

Та-да!

Чтобы транскрибировать ещё один файл в том же окне, достаточно выполнить шаг 5.
